# Disturbance-Chapter-Code & files necessary to reproduce disturbance chapter results! 
- Pipeline code: goes through cleanup process of the ASV files (phyloseq) using DADA2 bioinformatic pipeline
- Disturbance Chapter_GitHub.Rmd: goes through statistical tests and analyses of microbial data derived from the ASV files and the metadata collected.
- Statistical results include: beta dispersion and beta/alpha diversity anlyses, differential abundance code (structural zeros) and analyses of most relative abundant taxa, visualization via stacked barplots, boxplots, interaction plots ~ and environmental data statistics.
- Any dataframes needed to complete this analyses have been cleaned up and included
- should be able to rerun stats start to finish ~ and make plots ~ based on provided code and supporting files
- this is for a community composition study on microbial mat system in intertidal areas of the mid-atlantic coast, composition was analyzed using a combination of lab manipulations and in-situ field collection. ~ focus is on disturbance and temperature impacts on mat composition and function ~ this will be a chapter of my dissertation once completed and published.
