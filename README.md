# Disturbance-Chapter-Code & files necessary to reproduce disturbance chapter results! 
- Pipeline code: goes through cleanup process of the ASV files (phyloseq) using DADA2 bioinformatic pipeline
- Disturbance Chapter_GitHub.Rmd: goes through statistical tests and analyses of microbial data derived from the ASV files and the metadata collected.
- Statistical results include: beta dispersion and beta/alpha diversity ananlyses, differential abundance code (structural zeros) and analyses of most relative abundant taxa, visualization via stacked barplots, boxplots, interaction plots ~ and environmental data statistics.
